# Low Noise Amplifier (LNA) Design using Cadence Virtuoso at 2.45 GHz

## Project Overview

This project presents the design and analysis of a Low Noise Amplifier (LNA) using Cadence Virtuoso for RF CMOS applications operating at 2.45 GHz.

The objective is to achieve:

* Low input reflection
* High gain
* Proper impedance matching
* Better RF signal amplification
* Improved receiver front-end performance

This project is designed for wireless communication and RF front-end applications.

---

## Features

### RF CMOS LNA Design

The circuit is designed using:

* NMOS transistors
* Inductors
* Capacitors
* Biasing network
* Input/Output matching network

for optimized RF performance.

---

### S-Parameter Analysis

The following parameters were analyzed:

* S11 (Input Reflection)
* S21 (Forward Gain)
* S12 (Reverse Isolation)
* S22 (Output Reflection)

for proper RF verification.

---

### Gain Analysis

Different gain metrics were measured:

* Available Gain (GA)
* Power Gain (GP)
* Transducer Gain (GT)

to verify amplifier efficiency.

---

### Transient Analysis

Transient response was verified to ensure:

* stable signal amplification
* proper sinusoidal output response

## Final Results

### Operating Frequency

* Resonant Frequency = 2.45 GHz

---

### Input Matching

* S11 = -17.68 dB

Excellent impedance matching achieved.

---

### Gain Performance

* Available Gain (GA) = 19.59 dB
* Power Gain (GP) = 7.45 dB
* Transducer Gain (GT) = 7.36 dB
